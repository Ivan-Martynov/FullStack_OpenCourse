## Description

Following the lecture for [part3](fullstackopen.com/en/part3/) backend. Chose
to deploy the web site using [Render](https://render.com). For that purpose the
directory 'dist' is allowed in the .gitignore file (!dist/).
In order to create a web service in Render,
[this entire repository](https://github.com/Ivan-Martynov/FullStack_OpenCourse)
is used. Then the root is specified as necessary (e.g., for this application it
is set to part3/backend/lecturecode). Build command set to 'npm run build' and
start command to 'npm run start'
