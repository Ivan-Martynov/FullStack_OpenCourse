## Description

Exercise for part3 phonebook task.
The web site is deployed using [Render](https://render.com). The directory
'dist' is, therefore, allowed in the .gitignore file (!dist/).
In order to create a web service in Render,
[this entire repository](https://github.com/Ivan-Martynov/FullStack_OpenCourse)
is used. Then the root is specified as necessary (e.g., for this application it
is set to part3/backend/phonebook). Build command set to 'npm run build' and
start command to 'npm run start'
